Project Purpose

This project stemmed from wanting to do a personal project as well as improve my current skills.

I plan to use HTML/CSS to create a website. Then I will use some sort of database to store my Information. I will then use a tool to create dashboards.

In the current iteration, I plan to use MariaDB Database to store my information. Then I will use PowerBi to create the dashboards.

In the next iteration, I plan to use a different Database to store my information. Then I plan to user Tableau to create the Dashboards.

Finally, I decided that I will use different platforms to create this same project. This will be the first iteration with basic HTML/CSS and potentially some JavaScript. Then I will use a framework to create the second iteration. I may choose reactjs or php. This will require some research which I will put on this current website. Finally, I will use figma to design the website before utilizing their functionality to convert to react js code.
